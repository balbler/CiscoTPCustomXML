Use this folder to stash backup-bundle, roomControl xml and Macro files that need to be deployed to your endpoints.
The application will suck up these files into the correct XML format to deploy to your endpoints.

Backup bundles file currently in file is example only, make sure to delete and replace with your own.

In the case of the backup-bdunle deploymnet in CE9.3 a checksum will be created and allplication provides a
HTTP server to allow endpoints to download bundle package.

The proximity files are here as an working example. If you do not want this deployed to your endpoints make sure to remove before deployment.
